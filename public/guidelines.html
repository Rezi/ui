Vertical alignment (centering) of columns

Use .vcenter-container for a .row element and .vcenter for column element such as .col-md-5 in order to vertically align cols content

Beware that due to inline-blocks "gap" fix, columns has font size set to 14px by default (check the CSS to see why ;) )

ddd 

	<header class="row vcenter-container">
		<div class="col-sm-2 vcenter text-center">
			<img src="http://placehold.it/120x120" alt="..." class="img-circle">
		</div>
		<nav class="col-sm-8 vcenter">
			ddd
		</nav>
		<div class="col-sm-2 vcenter text-center">
			<img src="http://placehold.it/50x50" alt="..." class="img-circle">
			<img src="http://placehold.it/50x50" alt="..." class="img-circle">
		</div>
	</header>

Full height for columns

Use .full-height-cell-container for a .row element and .full-height-cell for column element such as .col-md-5 in order to set all coimns to 100% height. This is done by displaying rows as tables with real cells. Hence the responsiveness of the grid wont work.

Triange stuff

There is a cool way to create a triangle with only css, or even add a triangle after an element by CSS after


<style type="text/css">
	.components-with-arrow{
		position: relative;
		margin-bottom: 20px;
		border-bottom: 1px solid red;
	}


	.components-with-arrow:after{
		position: absolute;
		content:" ";
		height: 0;
		width: 0;
		left: 50%;
		bottom: -30px;
		border-style: solid;
		border-color: red transparent transparent;
		border-width: 15px;
		margin-left: -15px;
	}
</style>
